# Project-06
Get you and your friend a new model, then help him out with a favor.

# Features added
- Imported 3 character models + gun model
- Camera follows player
- Mixamo firing animation
- 3 targets with AI
- Multiple dialogue conversations with one npc
- Player score
- Probuilder buildings
- Gun for the player to use
- Avatar customization (choose your model and the npc's model) [bonus]
- Interact tooltip that displays when looking at something you can interact with [bonus]
